The functionality of functions in run_analysis.R
1. getSubject: Get a vector of subjects from training and test set
2. getActivity: Get a vector of activities from training and test set
3. getMean: Get vetor mean of variables in training and test set
4. getSd: Get vetor standard deviation of variables in training and test set
5. createTidyData: Create tidy data (data frame) from traning and test data set
6. createAvgTable: Create average table for all variables group by subject and activity
